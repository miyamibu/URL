URL Saver アカウント削除

最終更新日: 2026年6月27日

共有タグクラウドで作成したアカウントは、アプリ内またはメールで削除を依頼できます。通常の端末内保存 URL は、アプリ内で個別に削除できます。

アプリ内で削除する

  1. URL Saver を開きます。
  2. プロフィールまたは共有タグクラウドのアカウント画面を開きます。
  3. アカウント削除を選び、画面の案内に沿って削除を実行します。

メールで削除を依頼する

アプリを利用できない場合は、削除したいアカウントのメールアドレスを添えて miyamibu@privaterelay.appleid.com までご連絡ください。本人確認のため、追加情報をお願いする場合があります。

削除されるデータ

  • Supabase Auth のアカウント
  • 共有タグのメンバーシップ、招待、所有者移譲に関する情報
  • 削除対象アカウントだけが所有する共有タグのクラウドデータ

他の参加者がいる共有タグの所有者は、先に所有権を移譲する必要があります。端末内に保存された通常の URL、タイトル、メモ、タグは、アプリ内の削除操作で管理してください。
