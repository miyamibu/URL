URL Saver プライバシーポリシー

最終更新日: 2026年5月1日

URL Saver は、あとで開き直したい URL を保存し、必要な場合に共有タグの参加者と共有するためのアプリです。クローズドβでは広告表示、本物の課金、外部アナリティクスを行いません。

取得・保存する可能性がある情報

  • ユーザーが保存した URL、正規化した URL、表示用 URL
  • URL から取得したタイトル、説明文、サムネイル、投稿者名などのメタデータ
  • ユーザーが入力したタイトル、メモ、タグ
  • 共有タグの作成、参加、役割、招待、オーナー移譲に関する情報
  • Supabase Auth でサインインする場合のメールアドレス、認証に必要なアカウント情報

利用目的

保存した URL の一覧表示、詳細表示、重複判定、アーカイブ、削除猶予、メタデータ取得、共有タグ同期、招待リンクの発行と参加、アカウント管理のために利用します。

保存先と共有範囲

通常の保存 URL やローカルタグは端末内に保存されます。共有タグクラウドを有効にしている場合、共有タグに追加した URL と共有タグ参加情報は Supabase に保存されます。共有タグ内の URL は、その共有タグの参加者が閲覧できます。

URL メタデータ取得

保存された URL のタイトルやサムネイルを取得するため、アプリが対象ページや公開 oEmbed エンドポイントへ通信することがあります。取得できない場合でも、保存済み URL 自体は保持されます。

広告・トラッキング・課金

クローズドβでは広告を表示せず、本物の課金も行いません。Google Mobile Ads SDK は release ビルドに同梱しない方針です。外部アナリティクスや第三者クラッシュ収集サービスも現時点では導入していません。

削除方法

保存 URL やタグはアプリ内から削除できます。共有タグクラウドのアカウント削除は、アプリ内のアカウント削除導線、または公開アカウント削除ページから依頼できます。他の参加者がいる共有タグのオーナーは、先にオーナー権限を移譲する必要があります。
